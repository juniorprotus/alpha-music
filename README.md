# 🎤 Wax Kandle Alpha – Official Artist Platform

This project is a web platform built to showcase and promote the music, brand, and digital presence of **Wax Kandle Alpha**, a Kenyan Reggae and Dancehall artist.

## 🚀 Overview

The platform serves as a central hub for fans to:
- Discover music and videos
- Watch latest releases
- Explore artist content
- Connect and support the artist

Wax Kandle Alpha is a rising artist in the Kenyan reggae and dancehall scene, sharing music, performances, and visual content through platforms like YouTube.

## 🎶 Artist YouTube Channel

👉 https://www.youtube.com/@WaxKandleAlpha/shorts

The official channel features:
- Music videos & audio releases
- Shorts & freestyle clips
- Collaborations with other artists
- Behind-the-scenes content

## 🛠️ Tech Stack

This project is built using:
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js 
- Database: Supabase
